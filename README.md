# 同じ読みの かん字クイズ ― 仕様ドキュメント

小学1・2年生で習う漢字の「同音異義語／同じ読みの漢字」を、**例文を通じて意味を理解し、正しく使い分ける**ことを目的とした学習クイズアプリです。

- **ファイル**: `kanji-quiz.html`（単一HTMLファイル・依存ライブラリなし）
- **動作環境**: PC／スマホ／タブレットのブラウザ。ダブルクリックまたは下記コマンドで開く。
  ```
  open "kanji-quiz.html"
  ```

---

## 1. 目的・コンセプト

- 同じ読みでも漢字によって意味が変わることを、**やさしい例文の中で**学ぶ。
- 「読み」だけでなく「意味」で漢字を選ぶ力を育てる。
- 低学年向けに、ひらがな多め・大きいボタン・すぐ出る解説で構成。

---

## 2. 画面と機能

### スタート画面
- **学年セレクター**: `1年` / `2年` / `1・2年`（初期値 1・2年）。選んだ範囲で出題と選択肢が変わる。
- **モード選択（3つ）**
  1. **つかい分け** … 同じ音で意味がちがう漢字を文で見分ける（例: 会う／合う）
  2. **同じ読みのかん字さがし** … 「ほし＝星」「こえ＝声」など読みから漢字を選ぶ
  3. **ぜんぶまとめて** … ①②をシャッフルして出題

### クイズ画面
- 例文の空らんに入る正しい漢字を、選択肢から選ぶ。
- 回答すると**すぐに正誤＋全選択肢の意味の解説**を表示。
- **進ちょくバー／正解数／タグ（出題グループ名）**を表示。
- **🏠 HOME ボタン**（右上）… いつでも最初の画面へ。
- **← 前へ ボタン**（左上）… 一つ前の問題へ戻る。
  - 回答済みの問題に戻ると、自分の答え・正誤・解説がそのまま再表示される。
  - 戻っても**点数は二重カウントされない**（1問1回）。

### 結果画面
- 正解数 `◯ / △ もん`、星評価（⭐×3）、はげましコメント。
- **もう一回** / **さいしょの画面** ボタン。

### ランダム性
- 出題の**順番**（`shuffle(pool)`）と、選択肢ボタンの**並び**（`shuffle(q.choices)`）を毎回シャッフル。

---

## 3. データ構造（重要）

データは2層に分離し、**漢字の情報は1か所（KANJI）に集約**して二重管理を防ぐ設計。

### 3-1. `KANJI`（漢字マスタDB）
1文字ごとに学年・画数・音読み・訓読みを持つ唯一の正データ。

```js
const KANJI = {
  '海':{ g:2, s:9, on:['カイ'], kun:['うみ'] },
  // g: 学年, s: 画数, on: 音読み[], kun: 訓読み[]
};
```

- 1年生は**80字すべて**を収録。
- 2年生は**クイズで使う字を収録**（同じ書式で160字まで拡張可能）。
- 学年フィルタ・今後の解説表示はすべてこのDBから引く。

### 3-2. `BANK`（問題集）
例文と選択肢だけを持つ。**学年は書かず、`KANJI` から自動判定**する。

```js
{
  type:'onaji',                       // 'tsukai'（使い分け） or 'onaji'（同じ読み）
  tag:'同じ読み：かい',                // 画面上部に出る分類名
  sentence:'夏は @@ で およぐ。',      // @@ が空らん
  read:'うみ',                        // @@ の読み（ひらがな）
  answer:'海',                        // 正解（送りがな込み可: 例「会う」）
  choices:['海','回','貝'],           // 選択肢（answer を含む）
  meaning:{ '海':'うみ', '回':'まわる', '貝':'かい（貝がら）' }  // 各選択肢の意味
}
```

### 3-3. 学年フィルタの仕組み（`gradeOK`）
- 選んだ学年の上限まで（1年→1、2年/1・2年→2）の漢字しか**選択肢に出さない**（知らない字を見せない）。
- 単一学年（1 or 2）選択時は、**正解の漢字がその学年のもの**に限定。
- `1・2年` 選択時はすべて出題。
- ※`明ける/空ける`・`丸い/円い` のように **1年と2年の漢字をくらべる**問題は、性質上「1・2年」モードでのみ全問出る。

---

## 4. 収録コンテンツ

**合計 約163問**（2024年時点）。

### つかい分け（同音異義・33問）
1・2年生の漢字で成立する本物の同音異義ペア。

| 読み | ペア | 問題数 | 例 |
|---|---|---|---|
| あう | 会う / 合う | 16 | 友だちに会う／答えが合う・色が合う・力を合わせる |
| あける | 明ける / 空ける | 9 | 夜が明ける・つゆが明ける／席を空ける・コップを空ける |
| まるい | 丸い / 円い | 5 | ボール・お月さまは丸い／テーブル・おぼんは円い |

> 注: 1・2年生の漢字だけで作れる「同音異義」はこの3ペアが中心で、自然な例文の上限は約33問。さらに増やすには3年生以上の漢字（暑い/熱い/厚い、開ける、速い/早い 等）の解禁が必要。

### 同じ読みのかん字さがし（130問）
同じ音（主に音読み）の漢字グループから、意味で選ぶ。

- **1年生グループ**: か(火/花/下)、せい(正/青/生)、き(木/気)、さん(山/三)、せん(川/千/先)、し(四/糸/子)、しょう(小/上/正)、そう(草/早)
- **2年生グループ**: せい(西/声/星/晴)、とう(冬/刀/答/東)、まい(米/妹/毎)、き(帰/記/汽)、ちょう(鳥/朝/長)、こう(口/行/高/光/考)、かい(海/回/貝)、しょう(小/少)、じ(字/耳/時)、か(夏/家/歌)、ぶん(文/分/聞)、や(矢/野/夜)、ばい(売/買)、り(里/理)、どう(同/道/読)、てん(天/点/店)、ち(地/池/知)、たい・だい(大/台/体/太)、ご(五/午/後/語)、う(右/雨/羽)、げん(元/言/原)、きょう(強/教/兄/京)、けい(形/計)、しゃ(車/社)、しん(心/新/親/森)、でん(電/田)、がん(岩/顔/丸)、こく(国/黒/谷)、しゅう(秋/週)、しょく(食/色)、と(戸/土/図)

各グループには複数の例文（おかわり問題）を用意。

## 5. 問題の増やし方（メンテナンス手順）

1. **新しい漢字を使う場合** → まず `KANJI` に追記。
   ```js
   '園':{ g:2, s:13, on:['エン'], kun:['その'] },
   ```
2. `BANK` に問題オブジェクトを追加（3-2 の書式）。`answer` は必ず `choices` に含める。
3. 動作前チェック（任意・推奨）: 「答えが選択肢に含まれる／選択肢の漢字がDBにある／全選択肢に意味がある」を確認。

---

## 6. これまでの変更履歴（要点）

- 初版: つかい分け＋同じ読みの2モード、絵文字フィードバック、星評価。
- タイトルを「同じ読みの かん字クイズ」に変更、ふりがな（ruby）を撤去。
- サブタイトルを「― 文を読んで、正しいかん字をえらぼう ―」に。
- 学年を選択式（1年/2年/1・2年）に。漢字を `KANJI` データベース化（1年80字＋2年使用字）。
- 🏠 HOME ボタン／← 前へ ボタンを追加。回答状態を問題ごとに保持。
- 問題を大幅増量（同じ読みの別例文を多数追加、使い分けを33問に拡充）。
- 出題・選択肢ともにランダム表示。

---

## 7. 今後の拡張アイデア

- 正解時に「画数・音読み・訓読み」を `KANJI` から表示（漢字の知識も同時に習得）。
- 2年生160字をすべて `KANJI` に投入し、読みグループをさらに拡充。
- 1回のプレイ数を「全問／10問」など選べるように。
- 効果音・イラスト・ふりがな表示・ヒント機能。
- （要相談）3年生以上の漢字を解禁して、定番の使い分けを大量追加。
